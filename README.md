<a id="overview"></a>

## 🚀 Overview

This repository hosts an interactive **PyMACS-generated AVPR1A molecular dynamics dashboard** as a public GitHub Pages site.

The dashboard summarizes and visualizes molecular dynamics results for **AVPR1A** across multiple species:

- **Human**
- **Mouse**
- **Rat**

The dashboard is intended as a lightweight, browser-accessible companion to PyMACS-based analysis workflows and associated manuscript, preprint, or supplementary materials.

It includes interactive visual summaries of:

- protein RMSD
- protein RMSF
- radius of gyration
- ligand RMSD, where available
- binding-site residue behavior
- ligand contact profiles
- interaction-type matrices
- cross-species comparisons
- data availability and summary metrics

<a id="how-this-dashboard-was-generated"></a>

## ⚙️ How this dashboard was generated

This dashboard was generated from a completed AVPR1A molecular dynamics analysis directory using the PyMACS cross-species dashboard mode.

Representative command:

```bash
python 00_Pymacs_X_analysis.py \
  --root . \
  --cross-species \
  --contact-hover-framewise \
  --max-hover-framewise-mb 250 \
  --outdir AVPR1A-Data \
  --open
```

The resulting output directory contained:

```text
AVPR1A-Data/
├── index.html
├── cross_species_ALL.html
├── family_HUMAN.html
├── family_MOUSE.html
├── family_RAT.html
├── assets/
├── data/
└── images/
```

Those files were then published through GitHub Pages from this repository.

<a id="updating-the-github-pages-dashboard"></a>

## 🔁 Updating the GitHub Pages Dashboard

When a new PyMACS dashboard export is generated, the GitHub Pages repository can be updated directly from the analysis server.

Example update workflow:

```bash
cd ~/pymacs-avpr1a

rsync -av --delete \
  --exclude='.git/' \
  --exclude='.gitignore' \
  /home/jxs794/MD_JOBS/Nahuel/AVPR1A-Data/ \
  ~/pymacs-avpr1a/

find . -name ".DS_Store" -delete
find . -name "__pycache__" -type d -prune -exec rm -rf {} +

grep -qxF ".DS_Store" .gitignore || echo ".DS_Store" >> .gitignore
grep -qxF "__pycache__/" .gitignore || echo "__pycache__/" >> .gitignore
grep -qxF "*.pyc" .gitignore || echo "*.pyc" >> .gitignore

git add .
git commit -m "Update AVPR1A dashboard data"
git push
```

Important:

```text
Always exclude .git/ when using rsync --delete into a Git repository.
```

Otherwise, `rsync` can delete the local Git metadata and turn the folder into a non-repository.

<a id="notes-for-viewers"></a>

## 🔎 Notes for Viewers

This repository is intended to make the AVPR1A dashboard easy to inspect from a browser.

Because the dashboard is static:

- no login is required
- no server is running behind the scenes
- no user data are collected by the dashboard itself
- all dashboard assets are served from this GitHub Pages repository
- CSV files are included directly in the repository for transparency

Some dashboard pages may be large because they include detailed interactive molecular dynamics traces and hover-enabled contact data. Initial loading may take longer on slower connections.

<a id="scientific-context"></a>

## 🧠 Scientific Context

AVPR1A, the arginine vasopressin receptor 1A, is a G protein-coupled receptor involved in vasopressin signaling. This dashboard provides a cross-species visualization layer for comparing simulation-derived structural and dynamic behavior across human, mouse, and rat AVPR1A systems.

The dashboard is designed to help users explore:

- whether global dynamics are conserved across species
- how ligand-bound systems differ from apo reference simulations
- which residues contribute to binding-site behavior
- how interaction profiles vary by ligand and species
- where residue flexibility changes are concentrated

This dashboard should be interpreted as an interactive analysis companion rather than a standalone mechanistic conclusion.
